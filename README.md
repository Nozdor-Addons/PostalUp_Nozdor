Этот аддон расширяет функциональность почтового ящика в World of Warcraft, добавляя инструменты для быстрого открытия, отправки и управления письмами, включая массовые операции и автозаполнение.

Ключевые особенности

Открытие всех писем одним кликом с опцией выбора типов (например, аукционы); Экспресс-отправка нескольких предметов или денег с автозаполнением; Список контактов с автодополнением имен (альты, друзья, гильдия); Блокировка трейдов во время работы с почтой; Отчет о полученных деньгах и копирование содержимого писем.
